A) Installation and Setup - top

After downloading EZINE theme package, please extract your download theme package and go to theme folder, unzip EZINE.zip folder and upload to wp-content/themes/ folder of your Wordpress site.

Now login to your Wordpress site and go to Appearance => Themes menu and activate EZINE theme.

If you want quick setup for your website, you can import our xml data sample that included in theme package that locate in sample data folder, here the step how to import sample data :

  1. Login to WP-admin
  2. Go to Tools => Import menu and choose Wordpress
  3. In upload field choose the xml sample data and upload it
  4. Now you need to configure EZINE theme options.

H) Using Wordpress 3.0 Menu Manager - top

EZINE Theme support for new Wordpress Menu System, there's two built in menu place in EZINE Theme, for main navigation and footer, but you can still use the native menu if you dont want use this features, if you want to use this feature, please below step

  • Creating Main Navigation Menu (Header section)

    Go to Appearance => Menus, first you need to create menu place for Main Navigation (Header Area), for more detail please follow this image :

    as you can see in image above, there's some step that should follow :

    1. Click screen option at the right top of the menu page, then check descrition option
    2. Create a menu for Main navigation and named it with main menu (you can named it with your own defined name)
    3. Select The pages that will be included in Main Navigation
    4. You can order the pages and include some pages as the child pages and add subtitle/short description for your main navigation pages
    5. Assign the main menu in Theme Location where will be placed in Main Navigation field
    6. and Save the menu!

  • Creating Menu for Category navigation (Left bar section)

    explanation of above image :

    1. Create a menu for category navigation (left bar menu) and named it with categorymenu (you can named it with your own defined name)
    2. Select Categories that will be included in Main Navigation
    3. You can order the categories as your category menu structure
    4. Assign the categorymenu in Theme Location where will be placed in Category Menu field
    5. and Save the menu!


  • Creating Footer Navigation Menu (Footer section)

    Go to Appearance => Menus, first you need to create menu place for FooterNavigation (Footer Area), for more detail please follow this image :

    as you can see in image above, there's some step that should follow :

    1. Create a menu for footer navigation and named it with footermenu (you can named it with your own defined name)
    2. Select The pages that will be included in Footer Navigation
    3. You can order the pages as your footer menu structure
    4. Assign the footermenu menu in Theme Location where will be placed in Footer Menu field
    5. and Save the menu!
